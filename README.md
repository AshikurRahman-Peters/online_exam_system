# Online Exam System (PHP + MySQL)

A simple online exam system built with **Plain PHP + MySQL**.

## Features

* Admin login
* User registration and login
* Create exams
* Add MCQ questions
* Add True/False questions
* Random question order
* Exam timer
* User exam result
* Review answers
* Admin dashboard
* User dashboard

## Technologies Used

* PHP
* MySQL
* HTML
* CSS
* Bootstrap
* JavaScript

## Installation

1. Download or clone the project
2. Move the project folder to your web server directory (`htdocs` for XAMPP)
3. Create a MySQL database
4. Import the SQL file from the `database` folder
5. Update database connection in the config/include file
6. Run the project in browser

## Default Setup

* Configure your database credentials in the project before running.

## Project Structure

* `admin/` → Admin panel files
* `user/` → User panel files
* `includes/` → Common includes and auth files
* `assets/` → CSS, JS, images
* `database/` → SQL file
